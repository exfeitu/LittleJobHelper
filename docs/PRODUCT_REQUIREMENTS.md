# 人事科办公助手（网页版）需求文档

## 一、产品定位

面向体制内人事科工作人员的个人办公效率工具，核心解决日常工作记录、待办管理、多部门对接、时效提醒、历史工作回溯等需求，优先开发网页版，支持多端同步，数据安全可导出，后续可考虑拓展小程序，聚焦人事科党建、组织人事、纪检、编制、档案管理等核心工作场景。

## 二、核心使用场景

1. 主线工作：撰写领导材料、制作台账、统计报表等大任务推进。
2. 突发任务：档案查询（退休、工龄补录）、临时表格填报等小任务处理。
3. 外出对接：前往组织部、编办、宣传部等单位交资料、盖章、领文件。
4. 工作回溯：领导询问过往工作，需快速检索、核对历史工作记录。
5. 任务管理：多工作群接收的任务统一收纳，避免遗漏，跟进任务进度。

## 三、整体布局需求（固定）

采用简洁办公风布局，操作便捷，结构清晰，具体如下：

- 左侧：固定侧边导航栏（所有页面通用）
- 顶部：全局搜索栏 + 用户名 / 退出按钮（所有页面通用）
- 中间：主内容区（根据导航切换不同页面）
- 右侧（可选）：今日待办小面板（快速查看当日核心任务）

## 四、详细功能需求

### 4.1 侧边导航栏（固定菜单）

1. 首页（今日概览）：展示当日待办、当日工作记录、快捷操作。
2. 工作记录：带精确时间戳的工作日志管理。
3. 时间轴视图：日 / 周视图，直观展示每日 / 每周工作分布。
4. 待办任务：多级任务 + 细化信息管理（核心功能）。
5. 日历视图：待办与日程联动，展示任务时效。
6. 数据导出：工作记录、待办任务导出为 Excel。

### 4.2 首页（今日概览）

- 今日待办：展示当日所有待办任务，简要显示任务名称、提交部门、截止时间、提醒状态，未读提醒任务标红突出，点击可跳转至待办详情编辑。
- 今日工作记录：展示当天已记录的所有工作条目，每条包含精确时间戳、工作内容、工作类型标签。
- 快捷操作区：核心按钮 `开始记录当前工作`、`快速添加待办`，搭配简单输入框，实现快速录入，提升操作效率。

### 4.3 工作记录功能

- 记录内容：每条记录包含精确到秒的时间戳（开始 / 结束时间）、工作内容、工作类型标签（党建 / 人事 / 纪检 / 编制 / 档案 / 外出等可选），支持手动补充备注。
- 展示形式：列表形式展示，清晰直观。
- 操作功能：顶部搜索框、日期 / 类型筛选，底部 `新增记录` 按钮，支持单条记录编辑 / 删除。

### 4.4 时间轴视图功能

- 视图切换：支持日视图、周视图切换。
- 日视图：按小时纵向排列，每个时间段用色块标注对应工作，鼠标悬浮显示工作详情（时间、内容、类型）。
- 周视图：周一至周日横向排列，每天下方显示当日主要工作条目，支持展开查看全部。

### 4.5 待办任务功能（核心细化）

#### 4.5.1 任务结构

采用树形多级结构，支持一级、二级、三级待办，满足不同层级任务管理需求，示例：

- 一级任务：大工作（如：年度人事统计台账）
- 二级任务：子任务（如：统计在编人员信息）
- 三级任务：最小单元（如：核对 3 名工作人员工龄）

#### 4.5.2 任务字段（每条任务可选填写，避免冗余）

- 基础字段：任务名称、任务层级（一级 / 二级 / 三级）、截止日期、优先级（高 / 中 / 低，颜色区分）、完成状态（勾选框控制）
- 细化字段（新增）：
  - 提交部门：下拉选择 / 手动输入，预设常用选项（组织部、编办、宣传部、纪检组、档案局等人事科高频对接部门）
  - 联系人：输入姓名 + 可选联系方式（电话 / 微信），可保存至常用联系人列表，后续直接选择
  - 备注栏：多行输入，支持简单排版（换行、加粗），记录任务细节（如盖章要求、预约说明）
- 提醒字段（新增）：点击 `设置提醒` 按钮，配置个性化提醒，设置后显示提醒详情。

#### 4.5.3 操作功能

- 基础操作：勾选完成 / 取消完成、拖拽排序、编辑 / 删除任务。
- 筛选功能：可按提交部门、联系人、优先级、截止日期、提醒状态筛选任务。
- 便捷操作：常用部门、常用联系人可预设，减少重复输入；备注栏支持排版。

#### 4.5.4 提醒功能（新增）

- 提醒方式：优先网页内弹窗提醒，可选浏览器通知（避免错过重要任务）。
- 提醒时间：自定义设置（如截止前 1 小时、截止前 1 天、当日 8:30，适配不同时效需求）。
- 提醒频次：支持单次提醒、重复提醒（适配周期性任务）。
- 提醒标记：已开启提醒标注 `提醒已开启`，未设置标注 `未设置提醒`，快速区分。
- 提醒管理：可批量 / 单独开启、关闭、编辑提醒时间和频次。

### 4.6 日历视图功能

- 展示形式：月视图日历格子，有任务的日期标红点。
- 交互功能：点击某天，展开显示当日所有待办和工作记录；点击具体任务，弹出详情（含提交部门、联系人、提醒设置），方便外出前核对。

### 4.7 数据导出功能

- 导出范围：可选择今日 / 本周 / 本月 / 自定义时间段。
- 导出内容：工作记录、待办任务（含提交部门、联系人、备注、提醒状态等所有字段）。
- 导出格式：Excel，导出后可直接用于工作核对、汇报。

### 4.8 全局搜索功能

- 搜索范围：所有工作记录、所有待办任务。
- 搜索关键词：支持任务名称、提交部门、联系人、备注、提醒状态等，模糊匹配，快速定位。

## 五、数据安全与多端需求

- 多端登录：支持电脑、手机浏览器多设备登录，数据云端同步，不会因电脑清空、换设备丢失。
- 数据安全：无需复杂权限，单人使用，满足日常办公数据安全需求即可。

## 六、开发优先级

1. 第一期：工作记录（带时间戳）+ 多级待办（含提交部门、联系人、备注）+ 时间轴视图 + 全局搜索 + 多端同步。
2. 第二期：提醒功能 + 日历联动 + Excel 导出。
3. 第三期：小程序拓展（后续考虑）。

## 七、界面风格要求

简洁、清爽、实用优先，不花哨，操作简单，结构清晰，适合长期办公使用，减少视觉冗余，提升操作效率。

## 八、与当前原型对应的调整说明

为和本需求文档保持一致，当前原型应逐步向以下方向收敛：

- 布局改为左侧固定导航、顶部全局搜索、右侧今日待办小面板。
- 首页默认展示今日概览，中间主区域突出时间轴和今日工作记录。
- 时间轴保留纵向回溯能力，同时继续补充日 / 周切换。
- 待办模块继续补充提醒状态、任务详情编辑和快速新增能力。
- 数据导出从当前预览升级为真实 Excel 导出。
