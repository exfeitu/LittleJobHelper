# 产品路线图（基础版）

## Phase 1：Web 可用版

目标：让用户可以真实开始使用。

包含：

- 登录
- 云端同步
- 日程记录
- 待办管理
- 多级子任务
- 今日页
- 搜索
- Excel 导出

## Phase 2：Web 完整版

包含：

- 时间轴图表（日/周/月）
- 日历联动
- JSON 备份
- Excel 导入
- 附件上传

## Phase 3：移动端版本

包含：

- Expo 初始化
- 登录
- 今日待办
- 待办树
- 日程记录
- 搜索

## Phase 4：增强能力

包含：

- 周报/月报自动生成
- 工作耗时统计
- 多标签报表
- 智能提醒
- 语音输入

## 里程碑建议

### 里程碑 1
- 完成真实登录
- 完成待办与日程数据库接入

### 里程碑 2
- 完成 CRUD
- 完成今日页与搜索

### 里程碑 3
- 完成导出
- 完成时间轴

### 里程碑 4
- 完成 App 基础版
