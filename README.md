# Little Job Helper

体制内人事科办公助手：浏览器端工作回溯 + 待办管理工具。

## 当前功能

- **横向时间轴**：多天连续显示，滚轮缩放（0.16x~5.2x），视口固定 7 天，光标中心缩放，日分隔线，"今天"一键定位
- **快速记录工作**：模态弹窗补录，默认"过去2小时到现在"，快捷时间调整按钮，标签 chip 选择，支持关联待办
- **新建任务**：模态弹窗，标签 chip 选择，关联日程多选，优先级/状态/部门/联系人字段
- **多级待办树**：父子任务嵌套，状态自动提升，步骤进度展示，部门/联系人/备注
- **Event ↔ Todo 双向关联**：工作记录与待办任务相互关联，自动同步
- **文字日记视图**：当天工作记录按时间段展示
- **日历视图**：按天查看日程/待办，添加日程时自动生成关联待办
- **全局搜索**：关键词匹配 + 类型筛选
- **数据持久化**：LocalStorage 自动保存 + JSON 文件导入/导出
- **GitHub Pages 自动部署**：push master 即构建发布

## 技术栈

Next.js 16（静态导出） + TypeScript 5.8 + 手写 CSS + LocalStorage

## 运行方式

```bash
npm install          # 安装依赖
npm run dev          # localhost:10352
npm run build        # 静态导出到 out/
```
